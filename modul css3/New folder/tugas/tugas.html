Eksplorasi CSS Text Style

Kreasi dan Kombinasi Properti Text

Pada tugas ini saya mencoba menggabungkan beberapa properti CSS Text seperti text align, text transform, text spacing, text decoration, dan text shadow untuk menghasilkan tampilan teks yang lebih menarik dan nyaman dibaca.

Dengan pengaturan jarak antar huruf, jarak antar kata, serta jarak antar baris, teks menjadi lebih rapi. Selain itu, efek bayangan pada judul membuat tampilan terlihat lebih hidup dan modern. Beberapa teks yang tidak digunakan dapat diberi efek coret sebagai contoh penggunaan text-decoration.
